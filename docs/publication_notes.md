# Publication notes

This repository is associated with two SPICSCON 2026 conference manuscripts:

1. **Interpretable Lifestyle-Based Prediction of ECG-Derived Cardiovascular Phenotypes**
2. **Unsupervised ECG-Lifestyle Clustering for Early Cardiovascular Phenotyping in Young Adults**

The public repository intentionally contains code only. It does not include:

- raw ECG recordings
- lifestyle questionnaire spreadsheets
- participant identifiers
- merged participant-level feature tables
- manuscript-only generated figures or tables

The cleaned notebook is retained as a transparent record of the research workflow. The `src/ecg_phenotyping/` package provides a clearer modular structure for reuse and review.
